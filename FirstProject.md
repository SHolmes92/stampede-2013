# Introduction #

To check out the most basic robot code:
  * start NetBeans
  * Go to Team > Checkout...
  * Enter http://stampede-2013.googlecode.com/svn/trunk/MyFirstRobot into Repository URL
  * leave login and password blank
  * hit Next
  * check Scan for Netbeans Projects
  * hit Finish
  * after clicking done, an error message will appear regarding the certificate. click accept permanently
  * When done, click Open Project
  * You should see the MyFirstRobot project, expand it until you find SimpleBrain.java - this is the main source code of your robot!
  * Now you should be able to build and deploy this project
